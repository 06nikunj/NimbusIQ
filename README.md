# NimbusIQ

Intelligent AWS Infrastructure Monitoring & Optimization Platform built using Next.js, Express.js, PostgreSQL, AWS SDK, and cloud monitoring services.

## Features

* Real-time AWS monitoring
* EC2 monitoring
* Cost analytics dashboard
* Authentication system
* Infrastructure optimization
* Professional cloud dashboard

## Tech Stack

Frontend: Next.js, React, TypeScript

Backend: Express.js, Node.js

Database: PostgreSQL + Neon

Cloud: AWS SDK

## Installation

Frontend:

npm install

npm run dev

Backend:

npm install

npm run dev
